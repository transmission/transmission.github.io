These are third-party projects that other people have written for use with Transmission. Many of them are cool but don't look to us for technical support...

Apps and Utilities:

  • Transmission for NAS4Free
  • Browser-based remote controls: Kettu, Shift, Tr Web Control
  • Standalone remote controls: Transdroid, remote-cli, (video review), remote-gtk, remote-gui
  • Desktop remote controls: Transmission Indicator (GNOME Shell)
  • Android remote controls: Search Google Play
  • BlackBerry 10 remote controls: trans10
  • iOS remote controls: Monitor
  • RSS Tools: FlexGet
  • Nonfree remote controls: iControlBits
  • Browser add-ons: AutoTrans Reloaded (FF), .torrent-to-Transmission (Chrome)
  • XBMC plugin: Transmission-XBMC (github repo)
Resources
  • Remote control libraries: PHP-Transmission-Class, transmission_api (Ruby), transmissionrpc (Python), Transmission::Client (Perl)
  • 3rd Party Forums: NETGEAR ReadyNAS, QNAP, DSM-G600, DNS-3xx, NSA-220, Network Media Tank,
  • Erin Yueh has a writeup of porting Transmission to Android.
  • i2pTransmission is a project to get Transmission running on the I2P Anonymous Network.
Inactive / Abandoned Tools:
  • Transmission-Qt Win is a build of transmission-qt for Windows.
  • iTransmission is a port of Transmission for the iPad and iPhone.
  • Browser-based remote controls: p2p-gui
  • Instant Messaging-based remote controls: torrent-bot
  • Nonfree remote controls: Drivetrain (iOS), Transmission Mobile (Windows 7)
  • Other Apps: Transmission Supervisor is a helper application that adds some features to transmission-daemon.
  • Standalone remote control apps: remote-dotnet, remote-java, remote-symbian
  • Remote control libraries: Transmission Java API, P2P-Transmission-Remote (Perl), RubyTorrent, libtorrent-ruby
